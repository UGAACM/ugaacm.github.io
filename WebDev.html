Web Dev Group

General Description

The Web Development small group at the ACM is a targeted course aimed at showcasing key web development frameworks such as HTML5, CSS3, Javascript, C#, and ASP.NET as well as building the necessary skills to create and publish your own websites with key emphasis on the .NET MVC framework. Members will participate in building these skills through a comprehensive course and final project culminating in the construction of a basic, custom social media web site. We will work through various planned projects throughout the semester, though novel project ideas are welcome and encouraged!

The group meets Wednesdays at 7:00 PM Boyd Room 328. Until that time an email will be sent out with the meeting details, and you can check the ACM Calendar at here.

General Meeting Agenda

We will mainly be practicing on the W3Schools website (they have a built in code editor for different samples). The first meeting will be centered around getting everyone acquainted and set up with the right software and the HTML5 language. From there, we will dive into CSS, then Javascript and jQuery, C#, and ASP.NET. I will also be hitting on some other points, like SQL, PHP, XML, and Ruby on Rails (specifically to emphasize web security).
